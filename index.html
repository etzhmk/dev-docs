What You Should Already Know

Strukt CSS is a lightweight CSS framework inspired by neobrutalism and neo-minimalism. It was created as a free and open-source tool for developers, designers, and anyone who values clarity, structure, and style.

Before getting started, keep in mind that Strukt CSS is designed to be simple, lightweight, and visually distinctive.

Getting Started

To begin using Strukt CSS, follow these simple steps:

  1. Install Strukt CSS - Download the latest version from our repository or use a CDN link.
  2. Include the CSS file - Link the Strukt CSS file in your HTML document: 
<link rel="stylesheet" href="path/to/strukt.css">
  1. Start styling - Apply Strukt's predefined classes to your HTML elements for an immediate effect!

For more details, refer to our variables section to customize the framework to your needs.

Core Principles

Strukt CSS is built on the following core principles to ensure it remains lightweight, efficient, and easy to use:

  • Simplicity – The framework is designed to be as simple as possible, with minimal code and intuitive class names. You can quickly implement it without needing to understand complex configurations.
  • Modularity – Strukt CSS offers a modular structure, so you only need to use the parts of the framework that you need, keeping your project lean and optimized.
  • Responsiveness – The framework is built with mobile-first design principles, ensuring that your websites look great on any device, from mobile to desktop.
  • Consistency – Strukt CSS ensures a consistent design language across your project, with predefined spacing, typography, and color guidelines to streamline the design process.
  • Flexibility – While Strukt CSS provides basic styles, it’s also highly customizable to suit the unique needs of your project. You can override variables and classes to fine-tune the look and feel.
